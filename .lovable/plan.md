

# 登录页面实施方案

## 概述
创建一个分屏登录页面：左侧品牌展示区（渐变背景 + 同心圆装饰 + AI YOU 标识），右侧白色登录卡片（验证码/密码双 Tab 切换）。

## 新建文件

### `src/pages/Login.tsx`
分屏布局页面：

**左侧 (55%)**：
- 浅紫蓝渐变背景，CSS 同心圆装饰（border-radius 圆环）
- 左上角 "AI YOU" logo（蓝色圆点 + 粗体文字）+ 返回箭头
- 居中大标题 "● AI YOU"，副标题 "你的思维，从此多一个你。"
- 底部描述文字

**右侧 — 白色圆角卡片**：
- 标题 "欢迎使用 AI YOU"
- 两个 Tab："验证码登录" / "密码登录"（下划线激活态）
- **验证码 Tab**：手机号输入（+86 前缀）+ 验证码输入 + "获取验证码"按钮
- **密码 Tab**：手机号/邮箱输入 + 密码输入（眼睛切换）+ "忘记密码？"链接
- 共用：「记住我的登录状态」复选框 + 「立即登录」深色全宽按钮
- 底部："没有账号？立即注册" + 隐私协议链接

## 修改文件

### `src/App.tsx`
- 添加 `/login` 路由，import Login 页面

## 技术要点
- 纯前端模拟，无真实认证逻辑
- 点击「立即登录」后 `navigate('/')` 跳转主页
- 使用现有 UI 组件（Input, Button, Checkbox, Tabs）
- 同心圆用绝对定位 + border 实现
- 响应式：移动端隐藏左侧品牌区，仅显示登录卡片
