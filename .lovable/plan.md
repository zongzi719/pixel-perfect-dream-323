

# 移动端"分身"页 + "+"号AI记录功能开发

## 概述

基于设计稿，重构"分身"Tab页（当前 MobileMeetings 占位）为双Tab页面（AI CEO + 记忆库），并将底部"+"号改为弹出式 AI 快速记录面板。

## 页面结构

```text
分身页 (/m/meetings)
├── Tab: AI CEO (iPhone_17_279)
│   ├── 头像 + 用户名 + 标签 + AI学习数据
│   ├── 对齐率卡片 (Lv.X + 进度条 + 模型匹配度 + MBTI)
│   ├── 三列指标 (认知覆盖/核心关注/决策风格)
│   └── 四Tab内容区 (认知模型/语言风格/战略方法/决策逻辑)
│       └── 雷达图 + 描述文字
└── Tab: 记忆库
    ├── 子Tab: 历史文档 (iPhone_17_308/309)
    │   ├── 搜索框
    │   ├── 文档列表 (PDF/Word 图标 + 标题 + 预览 + 日期)
    │   └── 文档预览弹窗 (详情 + 查看更多/修改)
    ├── 子Tab: 用户记忆 (iPhone_17_307)
    │   ├── 搜索框 + 分类筛选 (全部/认知/语言习惯/决策风格/当前目标)
    │   ├── 待确认记忆卡片 (金色边框 + 取消/接受)
    │   └── 已确认记忆列表 (图标 + 分类 + 内容)
    └── 子Tab: 历史事项 (iPhone_17_299)
        ├── 搜索框 + 分类筛选 (全部/会议/安排/决策/其他)
        └── 按日期分组事项列表 (图标 + 标题 + 状态)

"+"号 AI记录 (iPhone_17_283)
├── 全屏弹窗 overlay
├── 极光背景 + 建议问题 chips
├── 底部语音录制按钮 + 计时器
├── 图片/链接附件按钮 + 发送按钮
└── AI分类结果页 (iPhone_17_300/301/302)
    ├── 日程安排 / 会议纪要 / 灵感笔记
    ├── 结构化内容展示
    └── 取消 / 保存 / 修改 三按钮
```

## 功能细节

### 1. AI CEO 页 (`AiCeoProfile`)
- 顶部栏：左"AI CEO" 右"记忆微调"按钮（金色星标）
- 头像 + 用户名 MOUMOU + 标签pills（创始人·科技创业者 / AI·互联网·SaaS）
- "AI学习数据 - 35份资料 · 2.5小时访谈"
- 对齐率卡片（金色边框）：Lv.3 + 当前水平标准 + 进度条（金色已完成/白色剩余）+ 预计提示
- 模型匹配度 65% + MBTI ENTJ（虚线分隔行）
- 三列指标底栏：认知覆盖-产品经验 / 核心关注-数据导向 / 决策风格-战略型思维
- 四个Tab：认知模型/语言风格/战略方法/决策逻辑
- 雷达图（Canvas 或 SVG）：四维度百分比，中心极光渐变
- 底部描述文字段落

### 2. 记忆库 (`MemoryLibrary`)
- 三个子Tab切换，深色圆角pill样式（选中白底黑字）
- **历史文档**：搜索框 + "近30天" + "只显示标题"开关 + 文件卡片（红色标题条 + 预览文字 + PDF/Word图标 + 日期 + ...菜单）
- **用户记忆**：搜索框 + 横向滚动标签筛选 + 待确认记忆（金色边框+倒计时+私密标签+取消/接受）+ 分类记忆卡片（认知/语言习惯/决策风格/当前目标，各带图标和颜色）
- **历史事项**：搜索框 + 标签筛选 + 按日期分组（今天/昨天）+ 事项行（图标+标题+状态圆圈：进行中/已完成）

### 3. AI记录弹窗 (`AiRecordModal`)
- 全屏半透明overlay，从底部滑入
- 顶部"AI记录" + X关闭
- 极光/蓝绿渐变背景卡片区域
- "今天有什么灵感/安排事项？告诉我，可以帮你生成"
- 建议chips + "换一批"
- 底部：图片按钮 + 链接按钮 + 发送按钮(↑)
- 语音录制大按钮（金色圆形 暂停/录制）+ 计时器
- 用户输入后调用 AI（复用 chat edge function）进行分类

### 4. AI分类结果 (`AiRecordResult`)
- AI自动识别内容类型：日程安排 / 会议纪要 / 灵感笔记
- 日程安排：📅图标 + "已为你整理为日程安排" + 结构化卡片（标题+时间+待办列表+行动要点）
- 会议纪要：📋图标 + "已为你整理为会议纪要" + 结构化卡片（核心内容+待办事项+关键结论）
- 灵感笔记：💡图标 + "已为你整理为灵感笔记" + 结构化卡片（核心想法+待办事项+关键结论）
- 底部三按钮：取消（白色）/ 保存（金色）/ 修改（黑色）

## 技术要点

- **雷达图**：使用 SVG 绘制四维雷达图，中心渐变效果
- **AI分类**：复用 chat edge function，system prompt 指导 AI 输出 JSON 结构（type + structured content），前端解析渲染
- **数据**：当前使用静态 mock 数据展示 UI，后续接入真实数据
- **深色主题**：黑色背景 + 白色文字 + 金色（#C9A84C / amber）强调色
- **无数据库变更**：纯前端 UI 开发，mock 数据
