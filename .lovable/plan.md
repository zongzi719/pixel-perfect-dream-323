## 功能细节

### 1. 知识库主页 (`MobileKnowledge`)
- 顶部标题"知识库"(金色) + 右上角"..."菜单
- 搜索框（圆角深色背景）
- 横向滚动文件夹卡片：每个卡片显示堆叠文档缩略图效果 + 文件夹名称（战略资料/会议记录/产品文档），点击切换当前文件夹筛选
- 选中文件夹后显示子Tab：全部 / 我创建的 / 我分享的（pill样式）
- 文件列表区分两种类型：
  - **文档类型**：PDF/Word彩色图标 + 标题 + 日期 + ...菜单，"近30天"标题 + "显示正文"开关
  - **会议记录类型**：金色音频波形图标 + 会议标题(truncate) + "发起人@时间" + ...菜单
- 点击文档 → 打开 MobileFilePreview 弹窗
- 点击会议记录 → 导航到 MobileMeetingDetail 详情页

### 2. 文件预览弹窗 (`MobileFilePreview`)
- 底部弹出半屏弹窗，圆角顶部
- 文件信息卡：标题 + 预览文字 + 文件大小 + 日期(金色) + 标签 + Word/PDF图标
- "#已添加到xxx知识库"提示
- "预览模式"标题(金色竖条) + 结构化内容（会议概览/核心决策/待办事项等）
- 底部固定：查看更多(白色) + 修改(黑色边框)

### 3. 会议记录详情页 (`MobileMeetingDetail`)
- 顶部"< 预览 ..."导航栏
- 会议信息卡：标题 + 发起人@时间
- 音频播放器：进度条 + 当前时间/总时长 + 均衡器/后退15s/播放/前进15s/倍速
- 三Tab切换：转写 / AI纪要(金色下划线active) / 章节
- **AI纪要**内容：
  - 金色大标题（如"AI数字增长战略的核心价值"）
  - 思维导图样式：左侧竖线连接节点标签 + 右侧展开卡片（深色背景，标题+要点列表）
  - 结构化段落卡片：标题 + bullet要点（圆角边框卡片）
  - 不同章节用不同颜色左边框区分（金色/蓝色）
  - 两列布局的要点卡片（如"识别组织断点"四项2x2网格）

## 技术要点

- 纯前端 mock 数据，无数据库变更
- 文件夹卡片使用 CSS 堆叠效果模拟多文档缩略图
- 思维导图用 CSS flex + 竖线/连接线模拟树状结构
- 音频播放器纯 UI 展示（mock 状态）
- 深色主题 + 金色(#C9A84C)强调色
