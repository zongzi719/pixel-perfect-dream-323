## 功能细节

### 1. 主页 (`MobileProfile`)
- 右上角⚙️设置图标 → 导航到 /m/settings
- 用户头像(圆形) + 用户名 MOUMOU + 标签pills(创始人·科技创业者 / AI·互联网·SaaS) + AI学习数据
- **待处理事务**: 金色竖条 + "待处理事务" + 百分比(金色) + 进度条(金色/灰色) + "3/8"
- **今日重点事项**: 金色竖条标题 + todo列表，每项有文字+右侧圆形checkbox(未完成空心/已完成勾选灰色)，支持左滑显示编辑/删除按钮
- **今日任务**: 金色竖条标题 + 任务卡片(深灰背景圆角)，每个卡片含：金色标题 + 时间(时钟图标) + 子任务bullet + 优先级标签(核心=金色/重要=深金/次要=灰金) + 完成进度"2/3 完成"

### 2. 账户中心 (`MobileAccountCenter`)
- 顶部导航栏：← 返回 + "账户中心"
- 用户卡片(深灰)：头像 + 名称 + 标签 + AI学习数据 + 右上角编辑✏️图标
- **账号信息**区域：邮箱/公司/职位/行业/已使用(图标+label+value，左右对齐)
- **AI CEO绑定**区域：当前绑定名称/对齐率/状态(进度条+Lv.3)/最近更新
- 底部金色按钮"继续优化"

### 3. 设置页 (`MobileSettings`)
- 顶部导航栏：← 返回 + "设置"
- 分组菜单：
  - 通用：🔔 通知
  - 隐私与安全：🔒 修改密码 / 🗑️ 清除缓存
  - 关于：ℹ️ 关于本应用
- 每项圆角卡片，带 ChevronRight
- 底部固定金色"退出登录"按钮

### 技术要点
- 全部使用 mock 数据，深色主题 + 金色(#C9A84C)强调色
- 今日重点事项的左滑操作使用 touch 事件 + CSS transform 实现
- 设置页和账户中心作为独立路由页面(不在 MobileLayout 底部Tab内，需要独立全屏页面带返回按钮)
